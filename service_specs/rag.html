字段名

值

是否必需

描述

content-type

multipart/form-data

必填

请求数据格式

请求

响应格式

字段

值

是否必需

描述

business_code

int

必填

业务代码

message

string

必填

响应文本信息

data

Object

必填

响应数据

data.file_id

string

必填

文件唯一id

示例

发送请求

curl -X POST https://localhost:16688/aog/v0.2/rag/file \
  -H "Content-Type: multipart/form-data" \
  -F "file=@/path/to/your/file.txt"

返回响应

{
     "business_code": 50000,
     "message": ""，
     "data": {
         "file_id": "asdgasbdohaklndlak123"
     }
}

Retrieval 服务

请求格式

请求头

参见 请求头中的常见字段

请求

字段名

值

是否必需

描述

file_ids

Array of string

必填

使用哪些文件进行检索

model

string

可选

大语音模型

text

string

必填

需要检索的文本

响应格式

除了在 响应体中的常见字段 中定义的字段外，该服务在其响应 JSON 体中可能还有以下字段：

示例

发送请求

curl -X POST https://localhost:16688/aog/v0.2/rag/retrieval\
-H "Content-Type: application/json" \
-d '{
    "model": "qwen3:7b",
    "file_ids": ["asdgasbdohaklndlak123", "....."],
    "text": "......."
}'

返回响应

{
  "business_code": 50000,
  "message": ""，
  "model": "qwen3:7b",
  "content": ".........."
}
